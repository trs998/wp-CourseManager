=== YVTS - Course Manager ===
Contributors: Tristan Scott
Requires at least: 3.5
Tested up to: 4.9.8

== Description ==
Lists courses, levels and scheduled runs of those courses. Allows a student to select a course run at a specific date, or resits, or commissioned courses, and then allows them to submit an application form for that course. Gathers appropriate data and emails it to contact email address.
